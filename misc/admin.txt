backdoor to admin panel

make sure it is a new user
localhost:3000/accounts/createAdmin/zuhra
than login using
username: zuhra
password: 1

once logged in, you can see link "Admin Panel" on your profile page.
click, it will go to
http://localhost:3000/cm-admin
